# 🏭 Apex Dynamics - Manufacturing Corporate Template

製造業や工業系のトップ企業に特化した、ダイナミックで力強いコーポレートサイトテンプレートです。

**👉 [Live Preview](https://kotao-boop.github.io/corporate-manufacturing/)**

## ✨ 特徴
- 企業の技術力とスケールを伝える、全画面表示（Full-screen Hero）レイアウト。
- 画像がゆっくりと迫ってくるFramer風のシネマティックなアニメーション（Parallax Zoom効果）。
- 極太のタイポグラフィと、テーマカラー（赤）を効かせたコントラストの高いUI。
- AIが生成した超精密な機械部品の高解像度画像を搭載。
